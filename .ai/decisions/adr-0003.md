# ADR-0003: Default coordination mode is mixed editorial plus code

**Date:** 2026-04-20T01:17:33.243285+00:00
**Recorded by:** codex
**Tags:** money-minsky-policy-mismatch, coordination, hub

## Context

This draft is being finalized through both prose refinement and experiment-backed verification. Future sessions need a default work model.

## Decision

Use a mixed editorial-plus-code coordination pattern for this post: prose work, evidence review, and reproducibility work are all valid continuation paths, and the hub should keep them as separate but related tasks.

## Consequences

New sessions should claim the most relevant task rather than treating the post as prose-only or code-only. The .ai handoff note and current spec are the shared context for all three workstreams.

# ADR-0004: Canonical publish-readiness task is task-006

**Date:** 2026-04-20T01:18:17.779086+00:00
**Recorded by:** codex
**Tags:** money-minsky-policy-mismatch, task-queue, hub

## Context

An initial publish-readiness task was created with placeholder dependency ids. The hub rejected a direct transition from ready to blocked, so the stale task needs an explicit supersession record.

## Decision

Treat task-006 as the canonical publish-readiness task for the post. Task-004 exists only as a superseded stub and should not be claimed for work.

## Consequences

Future sessions should claim task-006, not task-004, when doing the final publish-readiness review.
